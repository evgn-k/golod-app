# AGENTS.md

## Назначение репозитория

Репозиторий содержит исходный код PWA-приложения "Счетчик голодания" — инструмента для отслеживания времени голодовки с визуализацией прогресса и мотивационными фразами.

## Основные файлы

- **index.html** — главная страница приложения
- **manifest.json** — конфигурация PWA
- **src/app.js** — основной JavaScript код (таймер, диаграмма)
- **src/motivation.js** — мотивационные фразы
- **src/styles.css** — стили (Tailwind CSS)
- **vite.config.js** — конфигурация сборки
- **tailwind.config.js** — конфигурация Tailwind

## Обязательные проверки

После любых изменений:

1. Проверить сборку: `npm run build`
2. Проверить отсутствие ошибок в консоли
3. Убедиться, что структура `dist/` адекватна

## Ограничения и риски

- Дата начала и окончания голодовки жестко задана в `src/app.js`
- Приложение использует локальное время браузера (не синхронизировано с сервером)
- Мотивационные фразы статичны (можно расширить)

## Правила изменений

- Коммиты на русском языке
- Изменения в конфигурации должны сопровождаться обновлением документации
- Добавление новых зависимостей — только после проверки необходимости
- После каждого логически завершенного этапа — `git push`

## Зависимости

- vite (сборка)
- tailwindcss (стилизация)
- chart.js (круговая диаграмма)
- autoprefixer (CSS-автопрефиксы)

## Базовая модель работы

1. Изменения делаем локально в `src/`
2. Проверяем через `npm run build`
3. Протестированные изменения пушим в `origin/main`
4. GitHub Pages автоматически задеплоит изменения
